# Sauce Labs Visual client library

This package provides a client and a CLI tool to interact with Sauce Labs Visual API.

## Installation

```sh
npm install --save @saucelabs/visual-snapshots
```

## Development

Build:

```sh
npm run build
```

Execute:

```sh
node build/index.js [command] [params] 
```

Run tests:

```sh
npx jest
```

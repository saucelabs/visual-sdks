# Sauce Labs Visual Snapshot CLI

This package provides a CLI tool to create Visual snapshots of a provided PDF document.

## Installation

```sh
npm install --save @saucelabs/visual-snapshots
```

## Development

Build:

```sh
npm run build
```

Execute:

```sh
node lib/index.js [command] [params] 
```

Run tests:

```sh
npm run test
```
